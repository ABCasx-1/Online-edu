# Online-edu  
> 本项目包括用户前台系统和后台管理系统。🤗


## 前端技术栈👇  

* HTML、CSS、JavaScript
* Vue
* Nuxt.js
* ElementUI
* Axios

## 后端技术栈👇  

* Spring、SpringBoot
* SpringCloud Alibaba系列组件
* MybatisPlus
* MySQL、Redis
* Swagger

## 联调及测试工具👇

* Apifox
* Natapp
